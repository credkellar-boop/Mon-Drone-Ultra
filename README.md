# Mon-Drone-Ultra
Mon-Drone-Ultra is an autonomous, AI-powered smart-drone featuring a dual-tier PX4 and ROS 2 navigation stack for Waymo-like autopilot capability. It integrates Gemini VLM for intelligent audio/vision processing, localized facial recognition for security tracking, a Google Maps mobile dashboard, and a stabilized 10–60" cinematic pico-projector.
